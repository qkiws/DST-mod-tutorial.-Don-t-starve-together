# 🦌 DST Modding Tutorial: Don't Starve Together

This repository contains basic information and resources to help you create your first mod for **Don't Starve Together**. This guide is focused on creating simple custom items.

## 🛠 Required Tools

To build a mod, you will need the following software installed:

1.  **Code Editor:** (e.g., VS Code, Sublime Text, or Notepad++).
2.  **Don't Starve Together Mod Tools:** Available in your Steam Library (Search for "Mod Tools" under the "Tools" category).
3.  **Spriter:** For compiling animations and managing `.scml` projects.
4.  **TEX Converter:** To turn your `.png` files into `.tex` and `.xml` atlases.

---

## 💡 Tips for Beginners

* **Learning Curve:** While you can use AI to help build your mod's logic, I highly recommend watching **Lukas's tutorials** first to understand the core basics.
* **Reverse Engineering:** The most effective way to create something powerful is to look at the source code of existing mods. Navigate to your Steam workshop folder and study how successful mods are structured.
* **Start Small:** Master simple item creation before moving on to complex character animations or world-gen modifications.
* **Check origin source code** All game scripts are in folder : dontstarve_steam.app/Contents/data/databundles in scripts.zip, feel free to check it
